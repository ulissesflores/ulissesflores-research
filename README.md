# Ulisses Flores Research

Research software, simulation studies, and citable computational artifacts for AI systems and complex environments.

This repository is the umbrella hub for my academic and research-adjacent work. It links the paper workspaces, released software artifacts, and citation-ready repositories that support my work in AI systems engineering, forecasting, resilience, and reproducible computational research.

## At a glance

- Scope: papers, software artifacts, datasets, and release-oriented research outputs
- Primary modes of work:
  - reproducible research software
  - simulation and systems modeling
  - citable computational artifacts
  - multilingual documentation where useful
- Operating model: lightweight hub here, full implementation in dedicated repositories

## Research tracks

- **AI systems and tooling**
  - CLI tools, LLM workflows, and developer-facing research software
- **Simulation and complex systems**
  - discrete-event models, platform resilience, and distributed-systems behavior
- **Cyber-financial resilience**
  - public-market observability, stress modeling, and non-stationary forecasting workflows
- **Scholarly software infrastructure**
  - citation metadata, archival releases, DOI-backed artifacts, and provenance discipline

## Featured artifacts

- [llm-contextizer](https://github.com/ulissesflores/llm-contextizer)
  Zero-dependency Python CLI for turning codebases into token-efficient contexts for LLM and research workflows.
  DOI: [10.5281/zenodo.18343438](https://doi.org/10.5281/zenodo.18343438)

- [mit508-techgrowth-des](https://github.com/ulissesflores/mit508-techgrowth-des)
  Discrete-event simulation of Kafka-Flink-Iceberg pipelines under PACELC and burst-load stress.
  DOI: [10.5281/zenodo.19244059](https://doi.org/10.5281/zenodo.19244059)

- [cyberfinancial-resilience-lrblstm](https://github.com/ulissesflores/cyberfinancial-resilience-lrblstm)
  Research software for cyber-financial resilience using Little's Law-inspired proxies and a Bayesian LSTM roadmap.
  DOI: [10.5281/zenodo.18275035](https://doi.org/10.5281/zenodo.18275035)

- [odisseo-signal-atlas](https://github.com/ulissesflores/odisseo-signal-atlas)
  Multilingual GitHub discovery engine for frontier engineering signals, ranked repository scouting, and reproducible markdown reports.

- [mirofish-citation-edition](https://github.com/ulissesflores/mirofish-citation-edition)
  Citation-focused archival edition of MiroFish with DOI, multilingual documentation, and scholarly metadata.
  DOI: [10.5281/zenodo.19189253](https://doi.org/10.5281/zenodo.19189253)

## What lives in this repository

- paper workspaces and manuscript-side materials
- release notes and editorial support files
- a stable navigation layer to the main software and citation artifacts

Current structure:

```text
papers/
  yape-cellular-autonomy/   manuscript workspace and release notes
```

This hub is intentionally lightweight. Larger software artifacts, simulation packages, and archival releases live in dedicated repositories where they can maintain their own versioning, test surfaces, and reproducibility workflows.

## Research principles

- reproducible software and versioned releases
- clear authorship, provenance, and citation practices
- public-facing documentation for technical and academic audiences
- separation between umbrella navigation and repository-specific implementation details
